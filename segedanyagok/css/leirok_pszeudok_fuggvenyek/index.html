Leírók

A leírókkal meg nevezzük vagy be azanosítjuk a weblap bizonyos elemeit amit késöbb specifikus stílusokkal CSS-ben, vagy funkciókkal JavaScript-ben vagye egyéb nyelvekben tudunk ellátni. A leírók szerepe konkrétan az elemek be azonosítása. Ehez a következő leírókat használjuk:

  • Osztályleírók (CSS-ben: .ClassNeve HTML-ben: class="ClassNeve")

    • Az osztályleíró vagy angolul class egy az azonosítóleíró-nál alacsonyabb priorítású leíró de még mindig képes felülírni az egyszerű elemleírókat.

    Ez egy kiemelt szöveg ezért egy osztályleíróval lett formázva.


    Példa kód részlet:

    HTML:
    p class="kiemelt"> Ez egy kiemelt szöveg ezért egy osztályleíróval lett formázva. /p
    CSS:
    .kiemelt {
    background-color: rgb(179, 42, 42);
    font-weight: bolder;
    size: 20px;
    text-align: center;
    }

  • Azonosítóleírók (CSS-ben: #IdNeve HTML-ben: id="IdNeve")

    • Az azonosítóleíró vagy angolul identifier az összes leírók közül az egyik legfontossabb. Felülírni kizárólag az univerzális leíró tudja CSS esetében vagy az inline CSS.

    Ez egy fontos szöveg ezért egy azonosítóleíróval lett formázva.


    Példa kód részlet:

    HTML:
    p id="fontos"> Ez egy fontos szöveg ezért egy azonosítóleíróval lett formázva. /p
    CSS:
    #fontos {
    color: rgb(179, 42, 42);
    font-weight: bold;
    size: 20px;
    }

  • Univerzális leíró (CSS-ben: *)

    • Az univerzális leíró az a leíró ami az odal összes elemére vonatkozik. Ez az összes többi leírót felül írja és ezt felülírni nem lehetséges.

    Erre a szövegre kizárólal az univerzális leíró hat.


    Példa kód részlet:

    CSS:
    *{
    margin: 0;
    box-sizing: border-box;
    padding: 0;
    color: white;
    font-family: 'Roboto Mono', sans-serif;
    }

  • Elemleírók

    • Az elemleíró a leírók legegyszerűbb fajtája. Ezeket általában csak akkor használjuk ha az adott elem csak egyszer fordul elő az oldalon vagy az előforduló azonos elemek mind ugyan olyan stílussal vagy funkciókkal rendelkeznek.


Ha CSS-ben több leíróra meg egyező stílust szeretnénk alkalmazni akkor ezt egyszerűen megtehetjűk ha a leírókat vesszővel elválasztva felsoroljuk. A példában egy bekezdésnek, egy h1-es címsornak, egy .kiemelt osztályleírónak és egy #fontos azonosítóleírónak ugyan azt a 179, 42, 42 RGB színkódú piros színt szeretnénk adni.


CSS:
p, h1, .kiemelt #fontos {color: rbg(179, 42, 42);}

A leírókat emellet CSS-ben a következő módon is felírhatjuk:


CSS:
.kiemelt > p {color: rgb(179, 42, 42);}

Ennek köszönhetően most az összes bekezdés ami egy .kiemelt osztálykijelölővel ellátot elemen belül van az az elöbb említett piros színű lesz.

Pszeudók

A pszeudóosztályokat és szpeudóelemeket abban az esetben használjuk ha egy adott elem környezetét vagy egy bizonyos hejzetét vagy egy bizonyos részét szeretnénk vizsgálni majd arra stílus beállításokat alkalmazni.

Mi a különbség a pszeudóosztályok és pszeudóelemek között? A pszeudóösztályok az elemek helyzetét vizsgálja a pszeudóelemek pedig egy bizonyos részét. A pszeudóosztályok jelölése ":" A pszeudóelemek jelölése pedig "::"


Most ezt nézzük meg a gyakorlatban: Alkalmazzunk a bekezdések első betűjére 20px-es betűméretet


CSS:
p::first-letter {font-size: 20px;}

Azonban pszeudóosztályok és pszeudóelemek nem csak egy elem első betűjét tudják vizsgálni hanem rengeteg mást is. íme a legfontossabbak:

  • :hover

    • A "hover" az elem azon állapota amikor a felhasználó ráviszi az elemre az egérmutatót. Ebben az állapotban ezzel a pszeudóosztállyal tudjuk a stílusát az elemnek megváltoztatni.


    CSS:
    p {color: black; transition: 0.4s}
    p:hover {color:blue; transition:0.4s}
    Én a :hover pszeudóosztállyal lettem formázva!

  • :nth-child()

    • Az An+B jelölést használja az elemek kiválasztásához egy testvérelemek listájából, a lista végétől visszafelé számolva.


    CSS:
    p:nth-child(even) { background-color: rgb(179, 42, 42);}

  • ::before

    • Az adott elem elé tudunk szimbólumokat, szöveget beilleszteni annélkül hogy az a szöveg része lenne.


    CSS:
    p::before { content: "§ "; font-style: italic; color: blue;}

    Ez egy példa bekezdés.

  • ::after

    • Az adott elem mögé tudunk szimbólumokat, szöveget beilleszteni annélkül hogy az a szöveg része lenne.


    CSS:
    p::after { content: " ℃"; font-style: italic; color: blue;}

  • ::first-letter

    • A szöveg első betűjének stílusát tudjuk változtatni annélkül hogy a szöveg többi része változna.


    CSS:
    p::first-letter { color: rgb(179, 42, 42); }

    Az első betűm piros.

Függvények

A függvényekkel értékekt tudunk kiszámítani, színeket vagy szövegeket tudunk manipulálni, vagy komplex színátmeneteket, alakazatokat és még sok mást létrehozni.


Íme a leggyakrabban használt függvényekre.


  • calc()

    • A calc() függvénnyel számításokat tudunk végezni amit a CSS szabály értékét befolyásolja.

  • rgb()

    • Az rgb() függvénnyel színeket tudunk megadni az RGB (Red Green Blue) színmodellből.


    CSS:
    p {
    color: rgb(179, 42, 42);
    }

    rgb(179, 42, 42) kódú piros vagyok

  • rgba()

    • Az rgba() függvénnyel szintén színeket tudunk megadni az RGB színmodellből de a sima rgb() fügvénnyel ellentétben ezzel már az áttetszőségüket is tudjuk változtatni az adott színnek.


    CSS:
    p {
    color: rgba(179, 42, 42, 0.5);
    }

    rgba(179, 42, 42, 0.5) kódú piros vagyok

  • hsl()

    • A hsl() függvénnyel színeket tudunk megadni az HSL (Hue Saturation Lightness) színmodellből.


    CSS:
    p {
    color: hsl(0, 62%, 43.3%);
    }

    hsl(0, 62%, 43.3%) kódú piros vagyok

  • hsla()

    • A hsla() függvénnyel szintén színeket tudunk megadni a HSL színmodellből de a sima hsl() fügvénnyel ellentétben ezzel már az áttetszőségüket is tudjuk változtatni az adott színnek.


    CSS:
    p {
    color: hsla(0, 62%, 43.3%, 0.5);
    }

    hsla(0, 62%, 43.3%, 0.5) kódú piros vagyok

  • url()

    • Az url() fügvénnyel egy fájl elérési utvonalát tudjuk megadni.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background-image: url(kep.jpg);
    background-size:cover;
    border-radius: 10px;
    background-repeat: no-repeat;
    background-position: center;
    }

  • linear-gradient()

    • Ezzel fügvénnyel az elem szín vagy háttérszínének tudunk egy lineáris színátmenetet beállítani.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: linear-gradient(45deg, rgba(87, 173, 199, 1), rgba(165, 83, 237, 1));
    border-radius: 10px;
    }

  • conic-gradient()

    • Ezzel fügvénnyel az elem szín vagy háttérszínének tudunk egy kúp színátmenetet beállítani.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: conic-gradient(rgba(87, 173, 199, 1), rgba(165, 83, 237, 1), rgb(89, 87, 199), rgb(237, 83, 198), blue);
    border-radius: 10px;
    }

  • radial-gradient()

    • Ezzel fügvénnyel az elem szín vagy háttérszínének tudunk egy radiális színátmenetet beállítani.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: radial-gradient(rgba(87, 173, 199, 1), rgba(165, 83, 237, 1));
    border-radius: 10px;
    }

  • drop-shaodw()

    • A drop-shadow() függvénnyel egy árnyékot tudunk alkalmazni az adott elemre.


    CSS:
    div {
    height: 150px;
    width: auto;
    background-color: white;
    border-radius: 10px;
    filter: drop-shadow(0px 5px 10px rgba(0, 0, 0, 0.541));
    }

  • blur()

    • A blur() függvénnyel elmosódást tudunk alkalmazni az adott elemre vagy elem hátterére.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background-color: white;
    border-radius: 10px;
    filter: blur(1px);
    }
