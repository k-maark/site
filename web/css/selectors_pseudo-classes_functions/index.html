Selectors, Pseudo-classes and Functions - CSS (+Examples)

Selectors

With selectors we name or identify certain elements of a webpage which we can later provide with specific styles in CSS, or functions in JavaScript or other languages. The role of selectors is specifically to identify elements. For this we use the following selectors:

  • Class Selectors (In CSS: .ClassName In HTML: class="ClassName")

    • The class selector is a lower priority selector than the ID selector but is still capable of overriding simple element selectors.

    This is highlighted text therefore it has been formatted with a class selector.


    Example code snippet:

    HTML:
    p class="kiemelt"> This is highlighted text therefore it has been formatted with a class selector. /p
    CSS:
    .kiemelt {
    background-color: rgb(179, 42, 42);
    font-weight: bolder;
    size: 20px;
    text-align: center;
    }

  • ID Selectors (In CSS: #IdName In HTML: id="IdName")

    • The ID selector or identifier is one of the most important of all selectors. It can only be overridden by the universal selector in CSS or by inline CSS.

    This is important text therefore it has been formatted with an ID selector.


    Example code snippet:

    HTML:
    p id="fontos"> This is important text therefore it has been formatted with an ID selector. /p
    CSS:
    #fontos {
    color: rgb(179, 42, 42);
    font-weight: bold;
    size: 20px;
    }

  • Universal Selector (In CSS: *)

    • The universal selector is the selector that applies to all elements on the page. This overrides all other selectors and cannot be overridden itself.

    This text is only affected by the universal selector.


    Example code snippet:

    CSS:
    *{
    margin: 0;
    box-sizing: border-box;
    padding: 0;
    color: white;
    font-family: 'Roboto Mono', sans-serif;
    }

  • Element Selectors

    • The element selector is the simplest type of selector. We generally only use these when the given element only appears once on the page or when all instances of identical elements have the same styles or functions.


If in CSS we want to apply matching styles to multiple selectors, we can simply do this by listing the selectors separated by commas. In the example, we want to give the same red colour with RGB colour code 179, 42, 42 to a paragraph, an h1 heading, a .kiemelt class selector and a #fontos ID selector.


CSS:
p, h1, .kiemelt #fontos {color: rbg(179, 42, 42);}

We can also write selectors in CSS in the following way:


CSS:
.kiemelt > p {color: rgb(179, 42, 42);}

Thanks to this, now all paragraphs that are inside an element with a .kiemelt class selector will be the aforementioned red colour.

Pseudo-classes

We use pseudo-classes and pseudo-elements when we want to examine the context or a certain state of a given element or a certain part of it and then apply style settings to it.

What is the difference between pseudo-classes and pseudo-elements? Pseudo-classes examine the position of elements whilst pseudo-elements examine a certain part. The notation for pseudo-classes is ":" and the notation for pseudo-elements is "::"


Now let's look at this in practice: Let's apply a 20px font size to the first letter of paragraphs


CSS:
p::first-letter {font-size: 20px;}

However, pseudo-classes and pseudo-elements can examine not only the first letter of an element but many other things. Here are the most important ones:

  • :hover

    • The "hover" is the state of an element when the user moves the mouse pointer over the element. In this state we can change the style of the element with this pseudo-class.


    CSS:
    p {color: black; transition: 0.4s}
    p:hover {color:blue; transition:0.4s}
    I have been formatted with the :hover pseudo-class!

  • :nth-child()

    • Uses An+B notation to select elements from a list of sibling elements, counting backwards from the end of the list.


    CSS:
    p:nth-child(even) { background-color: rgb(179, 42, 42);}

  • :first-child

    • The first-child defines the first paragraph or the first element within an element.

    CSS:
    p:first-child { background-color: rgb(179, 42, 42);}

  • ::before

    • We can insert symbols and text before the given element without it being part of the text.


    CSS:
    p::before { content: "§ "; font-style: italic; color: blue;}

    This is an example paragraph.

  • ::after

    • We can insert symbols and text after the given element without it being part of the text.


    CSS:
    p::after { content: " ℃"; font-style: italic; color: blue;}

    The optimal room temperature is 22.5

  • ::first-letter

    • We can change the style of the first letter of text without the rest of the text changing.


    CSS:
    p::first-letter { color: rgb(179, 42, 42); }

    My first letter is red.

Functions

With functions we can calculate values, manipulate colours or text, or create complex gradients, shapes and much more.


Here are the most commonly used functions.


  • calc()

    • With the calc() function we can perform calculations which affect the value of a CSS rule.

  • rgb()

    • With the rgb() function we can specify colours from the RGB (Red Green Blue) colour model.


    CSS:
    p {
    color: rgb(179, 42, 42);
    }

    I am red with code rgb(179, 42, 42)

  • rgba()

    • With the rgba() function we can also specify colours from the RGB colour model but unlike the plain rgb() function, with this we can also change their transparency.


    CSS:
    p {
    color: rgba(179, 42, 42, 0.5);
    }

    I am red with code rgba(179, 42, 42, 0.5)

  • hsl()

    • With the hsl() function we can specify colours from the HSL (Hue Saturation Lightness) colour model.


    CSS:
    p {
    color: hsl(0, 62%, 43.3%);
    }

    I am red with code hsl(0, 62%, 43.3%)

  • hsla()

    • With the hsla() function we can also specify colours from the HSL colour model but unlike the plain hsl() function, with this we can also change their transparency.


    CSS:
    p {
    color: hsla(0, 62%, 43.3%, 0.5);
    }

    I am red with code hsla(0, 62%, 43.3%, 0.5)

  • url()

    • With the url() function we can specify a file path.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background-image: url(kep.jpg);
    background-size:cover;
    border-radius: 10px;
    background-repeat: no-repeat;
    background-position: center;
    }

  • linear-gradient()

    • With this function we can set a linear colour gradient for an element's colour or background colour.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: linear-gradient(45deg, rgba(87, 173, 199, 1), rgba(165, 83, 237, 1));
    border-radius: 10px;
    }

  • conic-gradient()

    • With this function we can set a conic colour gradient for an element's colour or background colour.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: conic-gradient(rgba(87, 173, 199, 1), rgba(165, 83, 237, 1), rgb(89, 87, 199), rgb(237, 83, 198), blue);
    border-radius: 10px;
    }

  • radial-gradient()

    • With this function we can set a radial colour gradient for an element's colour or background colour.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background: radial-gradient(rgba(87, 173, 199, 1), rgba(165, 83, 237, 1));
    border-radius: 10px;
    }

  • drop-shadow()

    • With the drop-shadow() function we can apply a shadow to the given element.


    CSS:
    div {
    height: 150px;
    width: auto;
    background-color: white;
    border-radius: 10px;
    filter: drop-shadow(0px 5px 10px rgba(0, 0, 0, 0.541));
    }

  • blur()

    • With the blur() function we can apply blurring to the given element or element background.


    CSS:
    div {
    height: 250px;
    width: auto;
    max-width: 350px;
    background-color: white;
    border-radius: 10px;
    filter: blur(1px);
    }
